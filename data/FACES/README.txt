
# FACES Dataset
http://wiki.knoesis.org/index.php/FACES

The DBpedia dataset version used: English version 3.9

Ideal summaries that were generated by 15 users are in 'ideal_summaries' folder. Each entity summary has length 5 and 10 summaries marked
in each .xls file using 'x' mark.

The instances used are listed in the 'instances.txt' file. Each row in the file has DBpedia entity url, number of distinct properties, number of distinct features,
wikipedia url for the entity, general typing information to identify the entity. :-: mark is used to separate entries in each row.
